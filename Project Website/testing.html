Testing

How will we check our application works correctly?

How will we carry out testing?

We plan to carry out the testing phase when we start development during Term 2. We will be using a combination of manual and automated testing.

After looking into automated testing to make the testing process faster, we have decided on using Selenium. Selenium is an open source software testing framework for web applications. The advantages of using it are that it does not require to learn any testing scripting language by using its IDE.

Possible Tests

  • Toolbar shows drop down menus when each of the buttons are clicked
  • When an option is dragged and dropped, a graph with the corresponding data appears on the canvas
  • Correct data appears on graph
  • Graphs can be dropped onto one another and their data will merge
  • Application is functional on all the popular browsers
