User Interface

How will users interact with our application?

Using Human-Computer Interaction Principles in our UI

We have decided on our proposed UI using some elements of ‘Summary of Don Norman's Design Principles’.

Visibility

  • The user will see a dashboard consisting of a toolbar and canvas.
  • All of the functions will be a visible feature of the toolbar.
  • Graphs will have visible ‘flip’ buttons in each corner which when pressed will show the data visualised in tabulated form.

Feedback

  • When the user clicks on the toolbar to create a graph, the canvas will be highlighted to indicate to the user to drag and drop the graph onto the canvas.
  • When the user clicks on a segment of data from a graph, the canvas will be highlighted again so that the user knows dragging and dropping it onto the canvas will create a separate visualisation.

Constraints

  • If the user is not able to drag and drop an item onto the canvas, the canvas will appear grey for the duration of the mouse-hold to indicate this.
